## 🚀 **Getting Started**

### **Quick Start**

```bash
# Clone the repository
git clone https://github.com/yourusername/cival-dashboard.git
cd cival-dashboard

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Add your Supabase and Redis credentials

# Start development server
npm run dev
```

### **Environment Setup**

```bash
# Required Environment Variables
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
REDIS_URL=your_redis_url

# Optional AI Integration
OPENAI_API_KEY=your_openai_key
GEMINI_API_KEY=your_gemini_key
CLAUDE_API_KEY=your_claude_key
```

### **Railway Deployment**

```bash
# Deploy to Railway (auto-deploys on git push)
railway login
railway link your-project-id
git push origin main
```

## 🎮 **Usage Guide**

### **Creating Autonomous Agents**

1. **Navigate to Agents Tab** → Click "Create Agent"
2. **Select Strategy Type** → Choose from 5 technical analysis strategies
3. **Configure Parameters** → Set capital, risk limits, and preferences
4. **Enable Autonomous Features** → Turn on memory, learning, vault integration
5. **Deploy Agent** → Agent begins autonomous trading immediately

### **Multi-Strategy Farm Deployment**

1. **Select Multi-Strategy Farm** → Choose farm option in agent creation
2. **Automatic Deployment** → System deploys 45 coordinated agents
3. **Real-time Coordination** → Agents share signals and coordinate entries
4. **Performance Monitoring** → Track cross-strategy performance and optimization

### **Memory & Learning System**

- **Automatic Pattern Recognition** → Runs every 30 minutes
- **Performance Adaptation** → Real-time parameter adjustments
- **Learning Insights** → Generated insights from trading experience
- **Memory Export/Import** → Full backup and restore capabilities
